### 프로젝트 설명
2015년 7월에 열린 Hoseo gamejam에서 유아교육과 학생들과 협업해서 제작한 게임.<br>
돈의 개념을 아직 잘 모르는 어린이들에게 게임을 하면서 돈의 가치까지 알 수 있도록 도와주자 라는 목표로 제작.<br>
